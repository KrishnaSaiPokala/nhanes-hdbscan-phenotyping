# Data availability

This project uses only de-identified public-use data from the National Center for Health Statistics / Centers for Disease Control and Prevention National Health and Nutrition Examination Survey (NHANES).

## Discovery data

- NHANES August 2021-August 2023 public-use files
- URL: https://wwwn.cdc.gov/nchs/nhanes/continuousnhanes/default.aspx?Cycle=2021-2023

## Temporal replication data

- NHANES 2017-March 2020 pre-pandemic public-use files
- URL: https://wwwn.cdc.gov/nchs/nhanes/continuousnhanes/default.aspx?Cycle=2017-2020

## Repository data policy

Raw NHANES XPT files are not redistributed in this repository. Users should obtain raw source files directly from NCHS/CDC. This repository contains only code, aggregate derived tables, figures, manuscripts, and result-summary artifacts.

## Public-data ethics note

The manuscript is a secondary analysis of public, de-identified NHANES data. It does not include identifiable individual-level information and does not involve a prospective clinical protocol or interventional study.

# Release notes: v1.0.0-preprint

This release synchronizes the GitHub repository with the medRxiv-ready preprint package for **Stable label-free cardiometabolic biomarker phenotypes in US adults**.

## Included

- Final manuscript PDF
- Final supplementary material PDF
- Main figures and extended-data figures
- Curated tables
- Aggregate results JSON
- Documentation for leakage control, data availability, reproducibility, limitations, and release audit

## Scientific scope

The release supports reproducible audit of an exploratory, label-free cardiometabolic biomarker phenotyping analysis in public NHANES data. It should not be interpreted as a clinical decision tool or diagnostic classification system.
